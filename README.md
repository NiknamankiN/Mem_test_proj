# Qt Project
Данный проект Qt является частью программно-аппаратного обеспечения, предназначенного для функционального и параметрического тестирования микросхем памяти с последовательным интерфейсом.\n
Для запуска проекта необходима установленная библиотека VISA от National Instruments (https://www.ni.com/ru-ru/support/downloads/drivers/download.ni-visa.html#442805). Она используется для подключения источника питания и мультиметра, а также их удалённого контроля.\n
Комплекс состоит из ПК, микроконтроллера PIC18F4520, мультиметра DM3068, источника питания DP832A, а также объекта контроля - микросхемы памяти с последовательным интерфейсом I2C.\n
Программная часть для ПК состоит из нескольких классов:\n
chip_param - предназначен для хранения прочитанных из файла параметров тестируемой микросхемы памяти;\n
mainwindow - главное окно, в котором проверяется подключение по USB источника питания и мультиметра к ПК;\n
paraminp - окно, предназначенное для ввода параметров питания микросхемы, передачи данных параметров в источник питания, и выбора COM порта подключения микроконтроллера, указания файла с заполненными параметрами микросхемы памяти, а также проверки подключения микроконтроллера и микросхемы памяти к микроконтроллеру;\n
resultform - форма для вывода результатов тестирования, перед её появлением происходит обмен данными с микроконтроллером и непосредственно тестирование микросхемы памяти;\n
testwindow - форма с выбором методов функционального тестирования и параметрического тестирования и указания файла, в который будут записаны результаты тестирования.\n
Для работы проекта необходима версия Qt не ниже 5.1, так как для обмена по UART с микроконтроллером используется библиотека QSerialPort.
